--- BƯỚC 1: TẮT WINDOWS DEFENDER / REAL-TIME PROTECTION ---
1. Bấm phím Windows, gõ "Windows Security" và nhấn Enter.
2. Chọn mục "Virus & threat protection".
3. Nhấn vào "Manage settings" bên dưới dòng Virus & threat protection settings.
4. Chuyển trạng thái "Real-time protection" sang OFF.

--- BƯỚC 2: TẮT TƯỜNG LỬA (FIREWALL) ---
Để quá trình xác thực license với server nội bộ không bị chặn:
1. Vào Control Panel > System and Security > Windows Defender Firewall.
2. Chọn "Turn Windows Defender Firewall on or off" ở cột bên trái.
3. Chọn "Turn off Windows Defender Firewall" cho cả hai mục (Private & Public).

1. Giải nén file "Setup_and_Crack.zip" (Password: 123).
2. Chạy file "Install.exe" với quyền Administrator.
3. Sau khi cài đặt xong, chạy file "Activator.exe" để hoàn tất.
